# Bank Marketing Customer Analysis


This project analyzes a banking marketing campaign dataset to identify customers who are likely to subscribe to a term deposit. The analysis follows the CRISP-DM methodology including exploratory data analysis, preprocessing, feature selection, clustering, and visualization.

## Dataset
This project uses the **Banking Dataset – Marketing Targets** available on Kaggle.

Dataset link:  
https://www.kaggle.com/datasets/prakharrathi25/banking-dataset-marketing-targets

The dataset contains customer demographic information, financial attributes, and marketing campaign data used to predict whether a customer will subscribe to a term deposit.

Source: Kaggle

## Project Objectives
- Understand customer behavior in marketing campaigns
- Identify customer segments with high subscription probability
- Extract interpretable rules for marketing strategies
- Visualize campaign insights through dashboards

## Project Pipeline

1. Business Understanding  
2. Exploratory Data Analysis (EDA)  
3. Data Preprocessing  
4. Feature Selection  
5. Customer Segmentation (K-Means)  
6. Pattern Mining and Insight Extraction  
7. Visualization using PowerBI

## Key Techniques
- Feature Importance (Random Forest)
- Mutual Information
- Log Transformation
- Data Discretization
- K-Means Clustering
- Elbow Method
- Rule Mining (Support, Confidence, Lift)
